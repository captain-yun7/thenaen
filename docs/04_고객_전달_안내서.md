### 3. Firebase (푸시 알림용)

앱에서 복약/운동/병원 알림을 보내기 위해 Firebase 프로젝트가 필요합니다.

- [ ] Google 계정 준비 (Firebase 관리용)
- [ ] Firebase 프로젝트 생성 권한 위임 **또는** 직접 생성 후 아래 파일 전달:
  - `google-services.json` (앱 빌드에 필요)
  - `service-account.json` (서버 푸시 발송에 필요)
  - Firebase 프로젝트 ID

> Firebase 프로젝트 생성이 어려우시면, Google 계정 정보를 주시면 저희가 대행하겠습니다.
